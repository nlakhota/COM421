How to Organize and Analyze Research Data

Prerequisites

Required Software

  • Microsoft Excel
  • PASW (formerly SPSS) statistics software, or free trial download

Required Knowledge

  • The basic functions and tools of Excel
  • Descriptive statistics and inferential statistics
  • Research design and how to identify independent variables and dependent variables

Learning Objectives

This 30-minute lesson is intended for graduate and advanced undergraduate students undertaking original research projects. After completion, you will be able to:

  • Format raw research data in an Excel spreadsheet for efficient analysis
  • Run basic descriptive statistics in Excel
  • Prepare the document for statistical analysis in the software package PASW

Format Data for Analysis

Make sure the computer you are working on has Microsoft Excel installed. Open the document ToolBox_DataLesson.xls in Excel and follow along with the lesson.

Variable Names

In the Raw_Dataset spreadsheet, the name of each variable has been entered in the first row of each column:

  • Each variable name must be different from other variable names
  • The first variable (in column A) is a unique identifier
  • Variable names must start with a letter (not numbers or special characters)
  • Name variables intuitively (e.g., change 'use' to 'perc_ease_of_use')

Figure 1: Tutorial type, gender, experience, and task categorical values have been formatted as dichotomous variables.

Data Values

Data values, or rows, have been input for each subject in the experiment:

  1. Decide on input conventions and stick to them (e.g., change "female" to "F")
  2. Separate data into component values when possible by adding new columns
  3. Double-check for data entry errors
  4. Delete redundant columns after reformatting

Figure 2: Values for each variable are entered in a consistent format.

Dichotomous variables allow for easy isolation of characteristics for analysis. For example, separating experience levels allows testing whether subjects with low experience have perceived ease of use scores significantly different from those with medium and high experience levels.

Run Descriptive Statistics

Run descriptive (summary) statistics on the dataset and output results in a new spreadsheet:

  1. Go to Tools > Add-Ins... and enable Analysis ToolPak
  2. Select Tools > Data Analysis > Descriptive Statistics
  3. Enter the input range and check Summary Statistics

Figure 4: Excel will not run descriptive statistics with non-numeric characters, so variable labels must be re-entered.

Analyzing descriptive statistics may lead to new questions. For example, perc_enjoy is much higher on average than perc_usefulness, which also has a large variance. This suggests that some users may not have considered the system useful even though they enjoyed using it.

Run Inferential Statistics

Open PASW and upload the spreadsheet. (IIT labs in Stuart Building, room 112, have PASW. You can also download a 30-day free trial.)

Loading Data

  1. Select "Open an existing data source > More Files..."
  2. Set file type to Excel and open the file
  3. Select the Formatted_Data worksheet

Figure 5: Don't assume PASW doesn't run a given test. Many are located under broader statistical classifications.

Consult UCLA's Choosing the Correct Statistic to ensure the statistical analysis is appropriate for your research question(s), variables of interest, and associated data types.

Figure 6: For Kruskal-Wallis, an independent (free_play) and a dependent variable (perc_usefulness) are selected.

Figure 7: PASW allows users to export statistics to common document types.
